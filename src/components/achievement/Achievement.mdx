# Achievement

The Achievement component renders a list of achievements based on the achievements state.
Currently the achievement list is based on a json file (achievements-content.json).

<Props of={Achievement} />

## Basic usage

<Playground className="bg-black">
  <Achievement />
</Playground>

Pass in achievements list via props

<Playground className="bg-black">
  <Achievement achievements={[
    {
      title: 'my achievement',
      description: 'my desc',
      level: 0
    },
    {
      title: 'my achievement 2',
      description: 'my desc 2',
      level: 0
    },
  ]}/>
</Playground>
